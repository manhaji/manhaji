Features · what each role gets
Cohort heatmap Principal
Section × rubric-axis heat grid. Spot at a glance which sections lag on Written Arabic vs Analytical. Click cell → drills to that section's students.
Intervention tracker Advisor
Per-student log of what's been tried: meetings, tutoring assigned, parent calls, IEP updates. Each entry timestamped with author. Open items at the top.
Demographic breakdown Principal
Nationality, primary language, gender, fee bracket. Useful for the diversity report ISO sends to the regulator. Exportable CSV.
Risk score Advisor
Composite score (0-100) per student combining attendance, rubric trend, behaviour notes, fee status. Threshold triggers "review needed" flag. AI-explained: "Risk 62 — rubric dropped 0.4 + 3 absences + no parent contact since Jan."
Bulk parent comms Both
Select N students → "draft a parent message" → Manhaj generates personalised drafts using the 17-template catalog. Teacher reviews & approves before send. Tracks reply.
Renewal pipeline Principal
Visual funnel: invited (242) → opened (210) → confirmed (208) → cash-collected (174). Last-touch timestamp. Bulk-nudge non-responders.
Cohort timeline Both
Term-by-term view of every student showing rubric, attendance, behaviour. Spot when something turned.
Behavioural incidents Advisor
Tagged events (positive + negative). Threshold triggers AI: "3 incidents in 2 weeks for Khalid — propose a check-in?" Approve → schedules a meeting + drafts parent note.
Teacher feedback panel Advisor
For a single student: aggregated comments from each teacher this term. AI condenses. Useful before a parent meeting.
Peer-group comparison Advisor
"Layla vs her section average" + "vs same-grade peers nationally" (anonymized). Don't compare to named peers; aggregate only (safety rule).
Admissions inbox Principal
New applications with their documents, AI-scored against entry criteria. Approve / hold / decline with draft response letter.
Quick search Both
Typeahead "find me Aya Mansour" — works across name, parent name, student ID, section. Keyboard shortcut Cmd-K.
